# 주의
- 개발중인 기능입니다
- 간단한 테스트는 가능합니다.


## 개요
배송대행지를 이용할 때 타오바오 구매내역 자동입력기능을 지원하지 않을 경우, 조금 덜 불편한 입력을 도와주는 크롬 익스텐션입니다.

## 사용법
- 타오바오에서 물건을 구입합니다.
- 구매내역페이지에 들어갑니다.
- 원하는 조건을 조회한 후 크롬내 번역기를 통해 영문으로 번역합니다.
- 원하는 만큼의 구매내역을 드래그 앤 드롭하여 복사합니다.
- 익스텐션 실행 - 버튼 클랙 후 팝업 내에 표시되는 값들을 활용합니다.
